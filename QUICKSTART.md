## For Non-Technical Users

### Step 1: Download Everything (1 minute)
1. Download all files in this folder to your computer
2. Keep them together in one folder named "himalayan-escapes"

### Step 2: Setup EmailJS (5 minutes)
**Why?** So booking forms send emails to you automatically.

1. Go to [EmailJS.com](https://www.emailjs.com/) ‚Üí Sign Up (free)
2. Connect your Gmail account
3. Create an email template (copy from `EMAILJS-SETUP.md`)
4. Get 3 codes: Public Key, Service ID, Template ID
5. Open `index.html` in Notepad/TextEdit
6. Search for `YOUR_PUBLIC_KEY` and replace with your code
7. Search for `YOUR_SERVICE_ID` and replace with your code
8. Search for `YOUR_TEMPLATE_ID` and replace with your code
9. Save the file

**Don't want to use EmailJS?** No problem! The WhatsApp button works without any setup.

### Step 3: Deploy to Internet (2 minutes)

**Easiest Method - Netlify Drop:**
1. Go to [app.netlify.com/drop](https://app.netlify.com/drop)
2. Drag your entire "himalayan-escapes" folder onto the page

4. Get your live website URL! üéâ

## Optional Improvements

### Add Your Own Photos
1. Take photos of your actual trips
2. Resize them to 1200px wide (use [iloveimg.com](https://www.iloveimg.com/resize-image))
3. Compress them (use [tinypng.com](https://tinypng.com/))
4. Replace the Unsplash URLs in `index.html` with your photo URLs

### Connect a Custom Domain
1. Buy a domain (e.g., himalayanescapes.com) from Namecheap/GoDaddy
2. In Netlify dashboard ‚Üí Domain Settings
3. Add your custom domain
4. Follow the DNS setup instructions
5. Wait 24 hours for DNS propagation

### Track Visitors (Google Analytics)
1. Create Google Analytics account
2. Get tracking code
3. Add before `</head>` in `index.html`
4. See how many people visit your site
